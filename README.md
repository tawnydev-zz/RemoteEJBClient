# RemoteEJBClient
Client standalone pour réaliser des requêtes avec le projet EJB Remote.
<h2>Pré-requis:</h2>
<ul>
<li>JBOSS 7.1</li>
<li>JDK 1.7.0_79</li>
</ul>
<h2>Installation:</h2>
<ul>
<li>Ajouter le fichier jboss-client.jar du dossier /bin/client de JBOSS 7.1 au projet via le BuildPath</li>
</ul>
